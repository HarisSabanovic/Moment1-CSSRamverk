Bootstrap

Bootstrap är ett ramverk som jag tycker är väldigt användbart när det kommer till att snabbt bygga responsiva hemsidor. Det är väldigt smidigt med den stora mängden färdiga komponenter som knappar, tabeller och formulär. Detta gör utvecklingen mycket snabbare, exempelvis navigeringsmenyn som bootstrap gör javascript koden själv för att göra den mer responsiv samt att den fungerar vid mindre skärm enheter, vilket gör det enkelt att skapa en design utan att behöva skriva mycket CSS.

Dokumentation som finns till ramverket tyckte jag också är väldigt tydlgit och extensiv, och med det menar jag att det finns mycket information samt en bra guide om hur man använder ramverket. Jag hade inga problem med att få till tabellen och formuläret så att det såg ut som jag hade tänkt mig, användandet av klasserna ansåg jag var rätt tydligt och inte komplicerat.

Det som jag tycker är mindre bra med bootstrap har kanske egentligen inte så mycket med själva ramverket att göra utan handlar mer om CSS-ramverk i sin helhet. Eftersom det redan finns fördefinierade komponenter så blir det svårt att sticka ut designmässigt och alla hemsidor råkar se likadana ut, samt generiska.
